Rook Lite is an AI-powered Chrome extension that analyzes websites for
marketing, SEO, and business insights.

## Features
- Analyze current webpage
- AI-generated strengths, weaknesses, improvements
- SEO & social presence feedback
- Overall score (out of 10)
- Clickable analysis history
- Offline history via localStorage

## Tech Stack
- React + TypeScript
- Vite
- Chrome Extensions API
- FastAPI backend (separate repo)

## For Developers
```bash
npm install
npm run dev
```
